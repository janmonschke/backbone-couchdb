backbone-couchdb by janmonschke twitter: @thedeftone

A couchdb connector for backbone with support for real time changes. Check out the annotated source and the example.

Dependencies

  • jQuery
  • jquery.couch.js
  • Backbone.js
  • Underscore.js

Download

You can download this project in either zip or tar formats.

You can also clone the project with Git by running: 

$ git clone git://github.com/janmonschke/backbone-couchdb

get the source code on GitHub : janmonschke/backbone-couchdb
